Get User Data:


This is a POST route. Return all the data regarding a customer by id.

Get Type By Price:


This is a PUT route. Returns all the Devices that cost between the two parameters that are given, keys are - min_price and max_price, for example: /getTypeByPrice/40/100
https://targil1vod.herokuapp.com/getTypeByPrice
